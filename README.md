<h1>個人網頁</h1>

## 目前規劃
預計當作部落格使用，會使用vue-route轉跳，以及熟悉其他工具。
詳細內容大概如下
# 首頁
- 自我介紹
- 經歷
- 技能專程
- 最新作品集(會到作品介紹)
- 最新分享(部落格)
- footer

# 作品集
- 作品介紹
- 作品連結
- 留言區

# 部落格
- 更新時間&最後修改時間
- 介紹內容
- 留言區

# 登入與註冊
- 使用firebase的auth與store建立資料(可刪除)
- 忘記密碼的找回密碼
- 可以使用信箱登入以及google登入，如果是信箱登入需先認證才得正常使用

# 個人資訊(登入後顯示)
- 修改個人訊息
- 修改密碼

# 留言牆(登入後顯示)
- 留言哪篇部落格
- 留言時間&最後修改時間
- 留言者的名稱(暱稱) 預設email
- 留言內容

# 資料庫(管理者權限才能使用)
- 部落格與帳號管理處
- 查看留言
